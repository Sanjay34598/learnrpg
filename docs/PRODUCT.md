# Product Brief

## Product Name

LearnRPG

## One-Line Pitch

A gamified career-learning platform where learners build job-ready skills through practical missions instead of passive lessons.

## Target Users

- Students preparing for data analyst roles
- Beginners who struggle with lecture-first learning
- Career switchers building a portfolio
- Self-taught learners who want structure and progress tracking

## Core Problem

Many learners read tutorials but do not practice enough. They forget concepts because they do not use them in realistic tasks.

## Product Solution

LearnRPG turns skills into missions:

1. Choose a career path.
2. Pick a course from the catalog.
3. Attempt a mission.
4. Unlock the explanation.
5. Earn XP and track progress.

## MVP Scope

The current MVP includes:

- Static landing page
- Career path selection
- Course dashboard
- Mission workspace
- Choice, code, and reasoning missions
- XP and streak system
- Local player profiles
- Data Analyst job-prep catalog

## Non-Goals For This Version

- Backend authentication
- Real code execution sandbox
- Payment system
- Multiplayer features
- Instructor dashboard
- Cloud sync

## Success Criteria

The project is successful if a learner can:

- Open the site without setup.
- Pick the Data Analyst path.
- Understand what to learn for the job.
- Practice with interactive missions.
- Track progress across sessions.
- Use the project as a portfolio-quality GitHub repo.

## Future Product Ideas

- Real Python and SQL execution
- Cloud save
- User accounts
- Daily quests
- Leaderboards
- AI-generated hints
- Admin tool for course creation
- Exportable progress report
