# Catalog

LearnRPG includes a broad skill catalog designed around job readiness.

## Data Analyst Core

| Course | Purpose |
|---|---|
| Python | Programming foundations for analysis |
| Database | SQL and relational data basics |
| Advanced SQL | Windows, CTEs, cohorts, interview SQL |
| Excel | Formulas, pivots, cleaning, quick analysis |
| Pandas | DataFrames, filtering, grouping, merging |
| NumPy | Arrays, shapes, vectorized thinking |
| Statistics | Distributions, tests, uncertainty, sampling |
| Data Cleaning | Missing values, duplicates, validation |
| Visualization | Chart choice and visual communication |
| BI Dashboards | KPI dashboards, DAX-style measures, layout |
| Business Metrics | Funnels, retention, ROI, impact thinking |
| Cloud & Warehouses | Warehouses, pipelines, query efficiency |
| Storytelling | Insight writing and stakeholder communication |
| Capstone Projects | End-to-end portfolio project practice |
| Portfolio & Interviews | Resume bullets, STAR answers, project framing |
| Interview Drills | SQL questions, case reasoning, project walkthroughs |

## Mission Types

### Choice

The learner chooses from multiple options. Good for prediction, tool choice, and quick conceptual checks.

### Code

The learner types code or formulas. The offline checker validates whether key ideas are present.

### Think

The learner writes a reasoning answer before seeing the mentor answer. Good for business cases, debugging, storytelling, and interviews.
